# Входы и выходы по пунктам

В текстовых файлах везде помимо букв допускаются (и оставляются без изменений) пробелы, отступы и переводы строк.

1) Вход - режим (0 == шифровка, 1 == дешифровка), ключ и имя входного файла; выход - результат шифровки/дешифровки.
2) Вход - два имени файлов: с зашифрованным и расшифрованным текстом; выход - ключ.
3) Вход - имя файла; выход - все варианты расшифровки с ключами [0, 25].
4) Аналогично пункту 3, но выводятся только варианты, содержащие слова, найденные в dictionary.txt.
