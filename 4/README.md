# Практическое задание 4

- tea - программа для шифрования шифром TEA.

Файл шифруется/дешифруется шифром TEA. Режим шифрования - output feedback, в качестве вектора инициализации используется nonce, который генерируется как 64-битное число при шифровании, после чего в зашифрованном виде пишется в начало результирующего файла, и считывается в зашифрованном виде из начала файла при дешифровании.

Сборка - make

# Входы и выходы

- vernam: вход - режим (0 - шифровать, 1 - дешифровать), имена файлов с ключом, входным текстом и для вывода шифрованного текста; выход - шифрованный текст, выведенный в заданный файл.
